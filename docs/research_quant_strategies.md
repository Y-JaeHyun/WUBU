## 권장 개발 로드맵

### Phase 1: 인프라 + 밸류 팩터
- 데이터 수집 파이프라인 구축 (주가/재무 데이터)
- 백테스팅 엔진 개발
- 밸류 팩터 단일 전략 구현 및 검증

### Phase 2: 모멘텀 + 이동평균 오버레이
- 모멘텀 팩터 전략 추가
- 이동평균 마켓 타이밍 오버레이 적용
- 밸류 vs 모멘텀 성과 비교

### Phase 3: 멀티팩터 + 듀얼 모멘텀
- 퀄리티 팩터 추가
- 멀티팩터 모델 (3팩터 결합)
- 듀얼 모멘텀 자산배분 전략

### Phase 4: 고도화
- 리스크 패리티, 저변동성 전략
- ML 기반 팩터 타이밍 실험
- 실전 매매 시스템 연동

## 한국 시장 데이터 소스

| 데이터 종류 | 소스 | 비고 |
|-----------|------|------|
| 주가/시세 | KRX, pykrx, FinanceDataReader | 무료 |
| 재무제표 | DART OpenAPI | 무료 API |
| 투자자별 매매동향 | KRX 정보데이터시스템 | 외국인/기관 수급 |
| 공시 | DART 전자공시시스템 | API 제공 |
| 거시경제 | 한국은행 ECOS, FRED | 금리, 환율 |

## 초기 구현 추천 전략 (Top 5)
1. **밸류 팩터** - 인프라 구축 기초, 한국 시장에서 가장 검증됨
2. **모멘텀 팩터** - 밸류와 낮은 상관관계, 주가 데이터만으로 구현
3. **듀얼 모멘텀** - ETF 기반 자산배분, 즉시 실전 적용 가능
4. **멀티팩터 모델** - 1,2의 자연스러운 확장
5. **이동평균 마켓 타이밍** - 기존 전략의 리스크 관리 오버레이
