## 2. 각 전략 상세 분석

### 2.1 ASY (Advanced Shareholder Yield)

**개념**: 배당 + 자사주 소각에 기반한 주주환원 신호

**파일**: `/mnt/data/quant-dev/src/strategy/advanced_shareholder_yield.py`

**주요 설정**:
- 포트폴리오 규모: 상위 15종목 (기본값)
- 배당 가중치: 0.4
- 자사주 소각 가중치: 0.6 (강력한 신호로 취급)
- 최소 시가총액: 1,000억원

**핵심 차별점**:
- 단순 배당 수익률이 아닌 자사주 소각을 명시적으로 가중
- Value-Up 정책(2025~) 하에서 구조적 재평가 신호
- 일본의 유사 개혁(2014-2016) 사례에서 significant alpha 입증

**데이터 요구사항**:
- **주요**: pykrx 배당 수익률 데이터 (이미 수집 중)
- **선택**: DART 자사주 소각 데이터
  - 미제공 시 배당수익률 기반으로 fallback

**기대 IC**: 0.08 ~ 0.12

### 2.2 RMOM (Residual Momentum)

**개념**: 시장 베타를 제거한 개별 종목의 idiosyncratic momentum

**파일**: `/mnt/data/quant-dev/src/strategy/residual_momentum.py`

**주요 설정**:
- 포트폴리오 규모: 상위 20종목
- 회귀분석 기간: 252거래일 (1년)
- 최근 스킵 기간: 21거래일 (1개월, 단기 반전 회피)

**핵심 메커니즘**:
```
잔차 = 실제 수익률 - (베타 × 시장 수익률)
잔차 모멘텀 = 잔차의 누적 기여도
```

**장점**:
- 시장 모멘텀 크래시(MDD 큰 기간)에서 더 견고
- 지정학적 충격(Mar 2026 같은) 후 회복력 강함
- 개별 종목의 진정한 강도 포착

**데이터 요구사항**:
- **필수**: KOSPI 인덱스 가격 (이미 제공 중)
- **필수**: 개별 종목 일일 수익률

**기대 IC**: 0.05 ~ 0.09

**테스트 상태**: 코드 검증 완료, 백테스트 대기

### 2.3 VDS (Value-Up Disclosure Score)

**개념**: KRX Value-Up 공시 수준을 평가하는 정책 기반 신호

**파일**: `/mnt/data/quant-dev/src/strategy/value_up_disclosure_score.py`

**공시 점수 체계**:
| 점수 | 의미 | 설명 |
|------|------|------|
| 0 | 공시 없음 | Value-Up 미참여 |
| 1 | 공시만 함 | 구체적 목표 없음 |
| 2 | ROE/배당 목표 | 일반적 신호 |
| 3 | 구체적 일정 포함 | 강력한 신호 (경영진 의지 강함) |

**주요 설정**:
- 포트폴리오 규모: 상위 20종목
- 최소 공시 스코어: 1 (공시가 있는 종목만)
- 가중 방식: 공시 스코어 비례 (높은 스코어에 높은 비중)

**정책 배경**:
- KRX, 금융위 "Corporate Value-Up" 프로그램 (2025~)
- 목표: 한국 시가총액 KOSPI 비율 상향
- 결과: 정책 부합 종목에 구조적 재평가 유도

**데이터 요구사항**:
- **주요**: KRX Value-Up Disclosure Portal
- **보조**: DART 공시 (IR 정보, 투자 보고서)
- **미제공 시**: 모든 종목에 기본 스코어(0.5) 부여 (fallback)

**기대 IC**: 0.10 ~ 0.15 (가장 높은 신호)

**특징**: 정책 리스크 최소화 → 정부 정책이 지원하는 구조적 알파
